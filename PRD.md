## 执行摘要

AIGC-MAS (Multi-Agent Service) 是一个深度垂直服务于视频制作场景的多智能体服务平台。平台通过 AI 智能体协同工作，为短剧生产、企业视频营销、地方文旅宣传三大核心场景提供端到端的视频内容生产解决方案。

**核心价值主张**：
- **降本增效**：将视频制作周期从 30 天缩短至 3-7 天，成本降低 60-80%
- **专业品质**：10+ 专业角色智能体协同，确保输出质量达到商业级标准
- **灵活定制**：支持从脚本到成片的全流程自定义，满足不同行业场景需求

**目标客户**：
- MCN 机构、短剧制作公司
- 中小企业营销部门、品牌方
- 地方文旅局、景区运营方

## 1. 市场分析与产品定位

### 1.1 市场背景

**行业趋势**：
- AIGC 视频生成技术快速成熟，商用化拐点已至
- 短剧市场规模 2025 年预计超 500 亿元，年复合增长率超 50%
- 企业视频营销需求爆发，中小企业视频内容缺口巨大
- 地方文旅数字化转型加速，短视频成为文旅宣传核心载体

**痛点分析**：
| 角色 | 核心痛点 |
|------|----------|
| 短剧制片方 | 剧本创作周期长、分镜成本高、制作流程不透明 |
| 企业营销 | 视频制作贵、沟通成本高、迭代周期长 |
| 文旅部门 | 专业团队缺乏、内容产出慢、风格难统一 |

### 1.2 产品定位

**一句话定位**：一站式 AIGC 多智能体视频生产平台，让专业视频制作像点外卖一样简单。

**产品愿景**：成为全球领先的 AI 视频内容生产基础设施。

**产品使命**：用 AI 重构视频内容生产方式，让每个人都能轻松制作专业级视频。

### 1.3 目标市场与用户画像

**三大核心赛道**：

| 赛道 | 市场规模 | 客单价 | 目标客户 |
|------|----------|--------|----------|
| AIGC 短剧 | 500亿+ | 5000-50000元/部 | MCN、短剧公司、独立制片人 |
| 企业视频营销 | 2000亿+ | 2000-20000元/条 | 中小企业、品牌方、营销 agency |
| 文旅宣传 | 500亿+ | 10000-100000元/系列 | 文旅局、景区、地方政府 |

**用户角色**：
- **项目管理者**：创建项目、把控进度、验收成果
- **内容创作者**：调整脚本、优化分镜、把控创意
- **运营人员**：数据监控、资产管理、团队协作

### 1.4 竞争分析

| 竞品 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|-------------|
| 即梦/可灵 | 生成质量好 | 单点工具、无工作流 | 多智能体协同 + 全流程工作流 |

### 3.4 视频制作工作台（核心功能）

**产品定位**：视频制作工作台是平台的核心交互界面，采用节点式工作流编辑器，将脚本创作、分镜设计、视频生成、后期处理等环节以可视化节点的方式串联，用户通过拖拽、连接节点来构建完整的视频生产流程。

#### 3.4.1 视频工作台布局

**整体结构**：
```
┌─────────────────────────────────────────────────────────────┐
│  顶部导航栏（项目名称、模式、邀请有礼、视频详细、积分、订阅、通知、用户）  │
├──────────────┬──────────────────────────────────────────────┤
│  左侧项目列表  │                                              │
│              │           中央画布区域（节点工作流）               │
│  项目1        │                                              │
│  项目2        │        [脚本节点] → [分镜节点] → [视频节点]        │
│  项目3        │                                              │
│              │                                              │
│  [新建项目]    │                                              │
│              │                                              │
├──────────────┴──────────────────────────────────────────────┤
│  底部工具栏（缩放控制、节点添加、设置、资产、历史、导出等）            │
└─────────────────────────────────────────────────────────────┘
```

**核心区域说明**：
1. **顶部导航栏**：显示当前项目名称、创作模式、积分余额、订阅入口、通知中心、用户头像
2. **左侧项目栏**：用户的项目列表，可折叠/展开，支持新建项目
3. **中央画布**：节点式工作流编辑器，黑色网格背景，支持平移（滚轮/中键拖拽）、缩放
4. **底部工具栏**：缩放控制、添加节点、视图重置、资产库、生成历史、设置、导出

#### 3.4.2 节点类型体系

| 节点类型 | 图标 | 功能描述 | 输入 | 输出 |
|---------|------|---------|------|------|
| 短视频脚本 | 📝 | 创建和编辑短视频脚本，包含场景描述、台词、时长等 | 无 | 脚本数据 |
| 分镜镜头 | 🎬 | 基于脚本生成/编辑分镜，可设置镜头数量、风格、参考图 | 脚本数据 | 分镜数据 |
| 视频生成 | 🎥 | 基于分镜描述生成视频，可配置模型、时长、分辨率等参数 | 分镜数据/提示词 | 视频文件 |
| 图片生成 | 🖼️ | 生成参考图片或角色形象图 | 提示词 | 图片文件 |
| 音频配音 | 🎵 | 生成配音或背景音乐 | 文本/脚本 | 音频文件 |
| 素材上传 | 📤 | 上传本地素材到项目中 | 无 | 素材数据 |
| 历史生成 | 📜 | 从历史生成记录中选择复用 | 无 | 素材/视频数据 |

#### 3.4.3 节点操作

- **添加节点**：右键画布弹出菜单选择节点类型，或点击底部工具栏的添加按钮
- **移动节点**：鼠标拖拽节点标题栏移动位置
- **连接节点**：从节点输出端口拖拽到下一节点输入端口建立连接
- **选中节点**：点击节点选中，显示选中边框和删除按钮
- **删除节点**：点击节点右上角的 X 按钮删除
- **编辑节点**：点击节点内容区域展开右侧属性面板，编辑节点参数
- **节点状态**：草稿（默认）、生成中（加载动画）、已完成（绿色标记）、失败（红色标记）

#### 3.4.4 视频生成设置

**尺寸比例**：
- 16:9（横屏，适合 YouTube、B站）
- 9:16（竖屏，适合抖音、视频号）
- 21:9（宽屏，电影感）

**分辨率**：
- 480P（流畅，节省积分）
- 720P（标准，推荐）
- 1080P（高清，专业品质）

**模型选择**：
- Seedance 2.0（稳定版，生成速度快）
- Seedance 2.0 Fast（快速版，更快但略降低质量）
- Seedance 2.0 Mini（轻量版，适合简单场景）
- Seedance 2.5（最新版，画质最佳，积分消耗高）

**时长选择**：
- 4秒 / 5秒 / 6秒 / 7秒 / 8秒 / 9秒 / 10秒 / 11秒 / 12秒 / 13秒 / 14秒 / 15秒

**风格选择**：
- 默认风格
- 电影感
- 动漫风格
- 国风水墨
- 赛博朋克
- 小清新
- 复古胶片
- 写实风格
- 支持自定义风格图片参考

**积分消耗规则**：
- 基础消耗 = 时长（秒） × 10 积分
- 模型系数：2.0（1.0x）/ Fast（0.7x）/ Mini（0.5x）/ 2.5（1.5x）
- 分辨率系数：480P（0.8x）/ 720P（1.0x）/ 1080P（1.2x）/ 4K（2.0x）

#### 3.4.5 创作模式

平台支持多种创作模式，每种模式有不同的工作流模板和智能体配置：

| 模式 | 适用场景 | 核心特点 | 典型流程 |
|------|---------|---------|---------|
| 短剧/漫剧模式 | 短剧制作、漫剧改编 | 连贯剧情，一键成片，支持多集管理 | 剧本 → 分镜 → 批量生成 → 剪辑合成 |
| 电影模式 | 微电影、广告片 | 高画质，长时长，精细控制 | 创意 → 剧本 → 分镜 → 逐镜生成 → 后期 |
| 电视剧模式 | 系列剧、连续剧 | 角色一致性，多集批量，剧情连贯 | 人设 → 剧本 → 分镜 → 批量生成 |
| 电商模式 | 电商短视频、口播视频 | 短视频矩阵、视频口播、动作参考 | 商品 → 脚本 → 分镜 → 批量生成 |
| 手搓模式 | 专业创作者 | 全手动控制，自由度最高 | 自由添加节点，自定义工作流 |
| 漫游模式 | 互动影视、游戏 | 互动式叙事，分支剧情 | 多分支剧本 → 分支生成 → 交互剪辑 |

#### 3.4.7 视频任务管理

**任务列表**：
- 显示所有生成中的视频任务
- 实时进度显示（进度条 + 百分比）
- 任务状态：等待中、生成中、已完成、失败
- 任务详情：模型、时长、分辨率、消耗积分、创建时间

**任务操作**：
- 查看生成结果（视频预览、下载）
- 重新生成（同样参数重新生成）
- 复制参数（复制到新的生成设置）
- 删除任务
- 批量管理（批量删除、批量导出）

#### 3.4.8 分镜生成

**分镜设置**：
- 镜头数量：1-10 个镜头可调
- 镜头比例：继承视频设置
- 风格：继承视频设置或自定义

**分镜编辑**：
- 每个镜头独立编辑提示词
- 上传参考图作为风格/构图参考
- 调整镜头时长
- 调整镜头顺序（拖拽排序）
- 增删镜头

**生成预演图**：
- 为每个镜头生成预览图（低分辨率快速生成）
- 确认预演图效果后再生成视频
- 节省高分辨率视频的积分消耗

#### 3.4.9 工作流执行

**执行方式**：
- 单节点执行：选中单个节点，点击生成
- 链路执行：从指定节点开始，自动执行后续所有连接的节点
- 全图执行：执行画布中所有未完成的节点

**状态流转**：
```
草稿 → 排队中 → 生成中 → 已完成
                    ↘ 失败 → （重试）→ 生成中
```

**进度通知**：
- 画布内节点实时状态更新
- 顶部通知提示（任务开始/完成/失败）
- 通知中心记录所有任务变动


#### 3.4.10 工作台入口 — 创作模式选择

用户从仪表盘/项目列表进入工作台前，需先选择**创作模式**，每种模式提供不同的工作流模板和智能体配置。

**模式选择界面**：
- 6种创作模式以卡片形式展示（2行×3列网格）
- 每张卡片包含：模式名称、适用场景描述、缩略图预览、"立即创作"入口按钮
- 卡片样式：圆角矩形，渐变遮罩覆盖缩略图，底部显示模式名称和描述

| 模式 | 描述 | 缩略图风格 |
|------|------|-----------|
| 短剧/漫剧模式 | 连贯剧情，一键成片 | 戏剧性人物场景 |
| 电影模式 | 服务于电影制作 | 电影级灯光场景 |
| 电视剧模式 | 服务于电视剧制作 | 群演/多角色场景 |
| 电商模式 | 短视频矩阵、口播、动作参考 | 产品展示场景 |
| 手搓模式 | 自由度最高，提示词驱动 | 艺术创作场景 |
| 漫游模式 | 互动影视游戏 | 沉浸式互动场景 |

**交互逻辑**：
- 用户点击"立即创作"→ 创建对应模式的新项目 → 进入空白画布工作台
- 已有项目可直接从左侧项目列表点击进入（自动识别该项目模式）

#### 3.4.11 空白画布引导状态

新建项目进入工作台后，画布为空白状态，需要引导用户开始创作。

**引导元素**：
- 画布中央显示引导文字："右击画布 添加短视频脚本 或 分镜镜头"
- 引导文字左侧附带光标图标
- 画布背景为点阵网格（24px间距），缩放时跟随变化
- 底部工具栏中央的"+"按钮也可触发添加节点面板

**操作入口**：
- 右键画布 → 弹出节点类型菜单（7种节点类型）
- 底部工具栏中央"+"按钮 → 弹出相同的节点类型菜单
- 两种方式功能一致，右键为高级用户快捷入口，"+"按钮为新手友好入口

#### 3.4.12 渐进式视频生成流程

这是平台核心的业务差异化能力。用户不必一次性配置所有参数生成最终视频，而是通过**渐进式预览**逐步确认每个环节，降低试错成本和积分消耗。

**完整流程链路**：

`
步骤1: 创建脚本节点
  → 用户输入/AI生成营销文案或剧本脚本
  → 确认脚本内容

步骤2: 生成预演图（可选但推荐）
  → 从脚本节点连接预演图节点
  → 支持本地上传预演图 或 AI生成预演图（消耗约3.8积分）
  → 获得低分辨率产品/场景预览，确认视觉方向

步骤3: 分镜拆解
  → 从预演图出发，选择镜头数量（1-10个）
  → AI自动将预演图拆解为多镜头分镜
  → 每个镜头生成独立的分镜缩略图和提示词

步骤4: 视频生成设置
  → 配置画面比例、分辨率、AI模型、时长、风格
  → 实时显示积分消耗预估
  → 提交生成任务

步骤5: 任务管理与结果查看
  → 异步生成，任务列表实时更新进度
  → 完成后可预览、下载、重新生成
`

**积分消耗对比**：

| 阶段 | 消耗 | 目的 |
|------|------|------|
| 预演图生成 | ~3.8积分/张 | 低成本确认视觉方向 |
| 分镜拆解 | 包含在预演图中 | 自动拆解为多镜头 |
| 视频生成 | 8.3积分/秒（1080P标准模型） | 高质量最终输出 |

**设计原则**：
- 每个阶段用户确认后再进入下一步，避免浪费积分
- 预演图环节为可选，高级用户可跳过直接进入分镜
- 所有积分消耗在操作前明确展示，用户确认后执行

#### 3.4.13 预演图节点

预演图节点是渐进式流程中的关键中间节点，用于在生成视频前低成本预览视觉效果。

**节点功能**：
- 显示当前预演图（图片画布区域）
- 两个操作按钮："本地上传预演图" 和 "历史上传预演图"
- 顶部工具栏："生成预演图 +3.8"按钮（标注积分消耗）
- 节点左右两侧有"+"按钮，用于添加相邻节点

**AI生成预演图流程**：
1. 用户点击"生成预演图 +3.8"
2. 系统基于关联的脚本节点内容生成低分辨率预演图
3. 预演图显示在节点画布区域
4. 用户确认视觉方向后，可进入分镜拆解环节

#### 3.4.14 分镜拆解弹窗

从预演图节点进入分镜拆解时，弹出分镜设置面板。

**弹窗内容**：
- 步骤指示器："① 拆解镜头数"
- 镜头数量控制器：数字显示 + 左右箭头调整（1-10）
- 两个操作按钮："镜头数量"（下拉选择）和"生成分镜"（主操作按钮，粉色）

**生成分镜后**：
- 预演图自动拆解为多个镜头缩略图
- 每个镜头有独立的提示词和时长
- 用户可编辑每个镜头的提示词和顺序
- 确认后进入视频生成环节
